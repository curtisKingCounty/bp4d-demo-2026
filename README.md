# Blueprint4D 2026 ASF Demo Project

This repository contains the Application Designer project for the ASF demo project. There are no changes to any delivered objects. Copy the BP4D_DEMO folder to a location accessible to your Application Designer client and import it from that location. There is nothing to build. 

Also included is the OpenAPI version of the API (BP4D_DEMO.openapi.json). This can be loaded into Apidog or any other OpenAPI-compatible tool to test with your installation. 

## Remember
Whichever account you wish to test with, ensure that one of the permissions lists assigned to the account is applied to all  the endpoints.
